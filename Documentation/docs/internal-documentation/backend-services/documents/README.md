# 📁 Drive

## description

**Drive** or **Documents** is the Nodejs drive implementation for twake, it contains drive items.

A **Drive item** can be:

- a file
- a directory
- the root folder
- the trash

## wording

**version** each file in the drive has a version associated to each update.
